Stack Technique CNRA Suite
La Suite CNRA est construite sur une architecture moderne, modulaire et cloud-native, conçue pour la haute disponibilité et la sécurité maximale des données de régulation audiovisuelle.

🖥️ Frontend

  • Next.js 16.2.9 (App Router)
  • React 19 (Server Components)
  • TypeScript 5 (strict mode)
  • Tailwind CSS v4
  • Radix UI (composants accessibles)
  • Recharts (visualisations)

🗄️ Backend / BDD

  • Supabase (PostgreSQL managed)
  • Row Level Security (RLS) activé
  • Supabase Auth (JWT)
  • Supabase Realtime (WebSockets)
  • 30+ tables SQL versionnées
  • Migrations Supabase CLI

🤖 Intelligence Artificielle

  • @anthropic-ai/sdk (Claude API)
  • Claude Sonnet (analyse contenu)
  • ElectroWatch: IA-assistant
  • KidsProtect: classification
  • AntiDeep: détection deepfakes
  • CnraAnalytics: insights direction

☁️ Déploiement

  • Vercel (hosting + CDN)
  • CI/CD automatisé
  • Preview deployments
  • Edge Functions
  • Variables d'env sécurisées
  • Monitoring Vercel Analytics

🧪 Qualité

  • Vitest (tests unitaires)
  • 64+ tests, 100% passants
  • ESLint + Prettier
  • TypeScript strict
  • Accessibilité WCAG 2.1
  • Design responsive mobile-first

📄 Export / Intégration

  • Export PDF (react-pdf)

  • API REST publique (MediaBase)
  • Webhooks configurables
  • Import CSV/Excel
  • Streaming (connecteurs live)

Sécurité : Row Level Security (RLS) activé sur toutes les tables contenant des données sensibles. Authentification JWT via Supabase Auth. Chiffrement des données au repos (AES-256). Accès API via clés rotatives stockées en variables d'environnement Vercel.
Architecture Système
Modèle monorepo modulaire : Chaque application est un module Next.js indépendant partageant une couche de services Supabase commune. Les 11 applications peuvent être déployées de manière autonome ou intégrée dans un portail unifié CNRA.

Couche Présentation

  • ▸ Next.js App Router (SSR + SSG)
  • ▸ React Server Components
  • ▸ Tailwind CSS v4 (design system)
  • ▸ Responsive mobile-first

Couche Métier

  • ▸ Server Actions Next.js
  • ▸ Services TypeScript typés
  • ▸ Intégration Claude API (IA)
  • ▸ Workflows automatisés

Couche Données

  • ▸ PostgreSQL (Supabase managed)
  • ▸ RLS + Auth JWT
  • ▸ Realtime WebSockets
  • ▸ Storage (fichiers, preuves)
